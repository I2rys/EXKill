## Installation
Github:

    git clone https://github.com/I2rys/EXKill
    
## Build
 1. Install [pkg](https://www.npmjs.com/package/pkg).
 

    npm i pkg -g

 2. Compile index.js to an exe file using pkg, for more information visit [me](https://www.npmjs.com/package/pkg).

    pkg index.js
